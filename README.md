# Tic Tac Toe Game

A console-based Tic Tac Toe game developed in C++.

## Features
- Two-player gameplay
- Dynamic board display
- Win detection
- Draw detection
- Replay option

## Technologies Used
- C++
- Arrays
- Loops
- Conditional Statements

## How to Run

Compile:

```bash
g++ tic_tac_toe.cpp -o game
```

Run:

```bash
./game
```
